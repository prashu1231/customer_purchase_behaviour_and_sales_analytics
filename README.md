Customer Purchase Behaviour & Sales Analytics
Project Overview

This project analyzes customer shopping behavior and sales trends using transactional retail data. The main objective is to identify customer purchasing patterns, product performance, subscription behavior, and revenue trends to support data-driven business decisions.

The project includes data cleaning using Python, exploratory data analysis, SQL-based business analysis, interactive Power BI dashboard development, and business recommendations.
